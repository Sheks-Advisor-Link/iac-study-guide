# IAC Study Guide — T. Van Heerden (2026)

A single-file, interactive HTML study guide for the **SAICA IAC / ITC** professional exams.
Exam dates: **23, 24 & 25 July 2026**.

## Features

- **📅 Interactive study calendar** (19 April → 25 July) — click any day for a 2-hour breakdown (Mon–Fri) or a two-topic weekend plan. Progress tracked in localStorage.
- **📘 ACC / 🔍 AUD / 💰 TAX / 📊 MAF summaries** — every high-yield IFRS, ISA, Act section and MAF concept, each with a plain-English "layman" explainer.
- **📊 Consolidations worked example** — full P Ltd / S Ltd walkthrough with pro-forma journals.
- **🔍 Substantive procedures by cycle** — assertion-linked procedure tables for 10 transaction cycles.
- **🎯 Weak-areas tab** — targeted drills for prior-period errors, IAS 38, IFRS 16, PIS, independence, s11(a), s24C, IFRS 3 goodwill, NOCLAR, NPV/WACC.
- **🃏 Leitner-box flashcards** — proper spaced-repetition (5 boxes, Again/Hard/Good/Easy ratings, auto-scheduled review).
- **📝 Self-marking quizzes** — 10 MCQs per subject + mixed paper.
- **🧮 Formula cheat sheet** — organised per topic with a 25-formula quick-reference table.
- **📄 Print / Save-as-PDF** — one click exports the whole guide in print-friendly formatting.

## Usage

Open `IAC-Study-Guide.html` in any modern browser. No server, no build, no dependencies — pure HTML + CSS + vanilla JavaScript. All progress (calendar ticks, Leitner scheduling, quiz scores) is saved to browser localStorage.
